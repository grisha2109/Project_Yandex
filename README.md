**"Spaceship adventures"**


Данная игра является простай аркадой, где игроку, управляющему космолётом, нужно собирать монетки и избегать астероидов.
Разработчики: **Григорий Ф. и Артем М**. 


_Движение и управление_
Игрок управляет космическим кораблём, который может двигаться только влево и вправо по нижней части экрана. Управление простое — стрелки клавиатуры или клавиши A и D. Корабль не может вылететь за границы экрана.
Сбор объектов
Сверху падают разные объекты, которые нужно или ловить, или избегать:
Космический мусор (хорошие объекты):
Обычный мусор: +10 очков
Ценный мусор: +50 очков (падает редко)
Нужно ловить, чтобы набирать очки
Астероиды (плохие объекты):
Маленькие: -1 жизнь
Большие: -2 жизни
Нужно избегать, чтобы не терять жизни
Бонусы (особые объекты):
* Щит: защищает от одного удара(или нескольких)
* Магнит: сам притягивает мусор к кораблю
* Замедление: замедляет все объекты на 5 секунд
Дают преимущества на время
Система жизней и поражения
У игрока есть 3 жизни. Каждое столкновение с астероидом отнимает жизнь. Когда жизни заканчиваются — игра заканчивается. Можно увидеть свой текущий счёт и количество оставшихся жизней в углу экрана.
Уровни сложности
Игра постепенно становится сложнее:
Каждые 1000 очков увеличивается уровень
Объекты начинают падать быстрее
Появляется больше астероидов
Реже выпадают бонусы


_Используемые технологии_
Программирование:
Python — основной язык программирования
Arcade — игровая библиотека для создания игр
Git и GitHub — для хранения кода и совместной работы
Графика и дизайн:
PNG формат — для всех картинок в игре
Мы создаём все графические элементы:
* Корабль игрока в виде космического челнока
* Космический мусор в виде обломков и деталей
* Астероиды разных размеров и форм
* Бонусы с узнаваемыми иконками
* Фон с звёздами и планетами
Хранение данных:
Текстовые файлы (.txt) — для сохранения рекорда
Разработка и тестирование:
PyCharm — среда разработки для написания кода
Встроенный отладчик — для поиска и исправления ошибок
Ручное тестирование — играем сами и ищем баги
